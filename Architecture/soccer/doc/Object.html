    get_and_clear_message() click to toggle source

    Helper method to obtain the message stored in session and clear it, so messages are only shown once.

    # File server.rb, line 26
def get_and_clear_message
  if session[:message]
    message = session[:message]
    session[:message] = nil
  end
  message
end
